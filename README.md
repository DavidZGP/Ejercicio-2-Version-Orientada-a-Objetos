# Ejercicio-2-Versi-n-Orientada-a-Objetos-POO-
Este repositorio contiene la versión orientada a objetos del proyecto Mini-Turtle.   Aquí se elimina el uso de variables globales y se encapsula todo el estado dentro de una clase llamada `Tortuga´.   Cada objeto creado mantiene su propia posición y permite dibujar figuras independientes usando métodos como adelante(), abajo() y reiniciar().
